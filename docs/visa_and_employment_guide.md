# オーストラリアでの雇用と学生ビザに関する重要事項

自身の ABN を使って友人とビジネスを行う場合、オーストラリアの法律（Fair Work Act）および移民法に基づき、以下の点に注意が必要です。

---

## 1. 友人（学生ビザ保持者）の就労制限

オーストラリアの学生ビザ (Subclass 500) には、厳しい就労時間の制限があります。

- **就労制限:** 原則として **2週間（Fortnight）で 48時間以内** です。
- **違反のリスク:** 制限時間を超えて働かせた場合、友人のビザが取り消されるリスクがあるだけでなく、雇用主（あなた）に対しても厳しい罰則や将来的なビザ支援への影響が出る可能性があります。
- **VEVO チェック:** 雇用主として、定期的に **VEVO (Visa Entitlement Verification Online)** で友人の労働権利を確認することが推奨されます。

## 2. 「従業員」か「外注（コントラクター）」か

友人に報酬を支払う際、主に2パターンの形式があります。

### A. 従業員 (Employee) として雇う
あなたが「雇用主」となり、友人を雇う形式です。
- **PAYG Withholding:** 給与から所得税を源泉徴収し、税務署 (ATO) に納める必要があります。
- **Superannuation:** 給与の **11.5%**（現在の率）を年金口座に積み立てる義務があります。
- **Workers' Compensation:** 業務中の怪我に備えた保険への加入が義務付けられます。

### B. 外注/請負 (Contractor) として依頼する
友人も **自分自身の ABN** を取得し、あなたに対して「インボイス（請求書）」を発行する形式です。
- **メリット:** 源泉徴収や年金の管理が不要（原則、本人の責任）になり、事務作業が楽です。
- **注意点 (Sham Contracting):** 実際には従業員のように働いている（シフトが決まっている、あなたの機材を使う、あなたの指示に完全に従う等）のに、税金を逃れるためにコントラクター扱いにすることは「偽装請負」として ATO から厳しく罰せられます。

## 3. 必要な手続き

- [ ] **TFN (Tax File Number) の確認:** どちらの形式でも必要です。
- [ ] **ABN 登録 (友人がコントラクターの場合):** 友人に ABN を取得してもらいましょう。
- [ ] **支払い記録の保管:** ATO の監査に備え、いつ、いくら、何の業務に対して支払ったかの記録を最低5年間保管する必要があります。

---

> [!WARNING]
> **「現金手渡し (Cash in hand)」の危険性**
> 記録を残さず現金で支払うことは、ATO（税務署）から脱税とみなされるリスクがあります。また、万が一友人とトラブルになった際、支払いの証拠がないことはあなたにとって大きな不利になります。必ず銀行振込等、記録が残る形での支払いを推奨します。
