# 雇用形態別のコスト・手残り比較（オーストラリア）

「どちらが手残りが多くなるか」という観点で、従業員として雇う場合と外注（コントラクター）として依頼する場合を比較します。

## 1. コスト比較シミュレーション

例：時給 $30 で 2週間（48時間）働いてもらった場合

| 項目 | 従業員 (Employee / Casual) | 外注 (Contractor / ABN) |
| :--- | :--- | :--- |
| **支払額 (BASE)** | $1,440 | $1,440 |
| **年金 (Super 11.5%)** | **+$165.6** (支払い義務) | $0 (原則不要 ※1) |
| **労災保険 (WorkCover)** | **+$20〜40** (加入義務) | $0 (本人の責任) |
| **事務コスト** | 会計ソフト代、申告手間 | $0 (振込のみ) |
| **合計コスト** | **$1,625〜** | **$1,440** |
| **あなたの「手残り」** | **少ない** | **多い** |

*(※1) 労働力のみの提供（Labor hire）とみなされる場合、コントラクターでも Super の支払いが必要になるケースがありますが、小規模なスタートアップでは外注扱いにすることが一般的です。*

---

## 2. なぜ「外注 (ABN)」が一番手残りが多いのか

1.  **付随費用の回避:** 従業員として雇うと、給与とは別に **約 12〜15% の追加コスト**（年金・保険・諸税）が確実に発生します。
2.  **管理コストの削減:** 従業員の場合、Single Touch Payroll (STP) という政府へのリアルタイム報告が必要になり、会計士に頼むか、有料のソフトが必要になります。
3.  **社会保障の負担:** 外注の場合、所得税の納税や保険の加入は「受け取り手（友人）」の責任になるため、あなたの財布からは出ていきません。

## 3. 注意点：友人の「手残り」とリスク

あなたが一番得をするのは「外注（ABN）」ですが、以下のリスクに注意してください。

- **友人の税金:** 友人が ABN で受け取った場合、年度末のタックスリターンで自分で税金を払う必要があります。その分を見越して、少し多めの時給を設定してあげるのが「友好的な」契約です。
- **偽装請負 (Sham Contracting) の回避:** 
  - 形式上は ABN（外注）でも、実態が「あなたの指示通りに動くボランティア/労働者」だと ATO（税務署）に判断されると、後から未払いの年金や罰金を請求されるリスクがあります。
  - **対策:** 「何時から何時までここにいろ」という命令ではなく、「この作業をいくらでやってくれ」という **業務委託の形** にすること。

## 4. 結論：一番賢い選び方

- **スタート時:** **外注 (ABN to ABN) 契約** が圧倒的に手残りが多く、事務も楽です。
- **友人に ABN を取ってもらう:** 友人が ABN を持っていれば、あなたは友人からの「インボイス（請求書）」を受け取り、銀行振込するだけで経費として正当に処理できます。

---

> [!TIP]
> **「手残り」を守るための経費処理**
> 友人に支払う報酬は、あなたのビジネスの「経費」になります。確定申告時に自分の売上から差し引けるので、あなたの所得税も安くなります。そのためにも、必ず **ABN 間の取引（インボイスの受領）** という形を整えましょう。
